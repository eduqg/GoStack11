# Gobarber Web - Gostack 11

Create project

```console
yarn create react-app gobarber-web --template=typescript
```

Icons

```console
 yarn add react-icons
```

To Forms

```console
  yarn add @unform/core @unform/web
  yarn add yup
  yarn add @types/yup -D
```

Animations

```console
  yarn add react-spring
```
